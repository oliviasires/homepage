Process

This project was created to visualize the Butterfly Effect through digital interaction. I designed each page to represent a different stage of growth, using layered media such as video, audio, GIFs, and imagery to gradually transform the environment. As the user navigates through the site, these elements become more complex and visually dense, mirroring how small actions can build into larger consequences. The progression from minimal visuals to a flourishing digital garden reflects the accumulation of choices and interactions over time.
